Administrador de Inventario

El entregable es una administracion de inventario en el cual se le añaden productos hasta que el ususario ingresa cero.
Tambien se puedan hacer las siguiente acciones:

  • crear una lista de productos con sus respectivos datos
  • agregar productos, filtrar y buscar por categoria
  • calcular el valor total del inventario
  • mostrar productos con menos de 5 elementos
